# SmartCart AI

## Project Overview

SmartCart AI is a console-based shopping cart management system developed using C++. It allows users to add, remove, and update items in their cart and generates the total bill with GST.
The project also includes a basic recommendation feature that suggests related items based on user selection.

## Features
1. Add items to cart
2. Remove items from cart
3. Update items
4. Generate total bill with 18% GST
5. Exit
Basic recommendation system

## AI Concept
This project uses a simple rule-based recommendation system.
Example:
Milk → Butter
Fruits → Water

## Technologies Used
1. C++
2. Console I/O

## How to Run
1. Compile the code:
    g++ main.cpp -o cart
2. Run the program:
./cart
