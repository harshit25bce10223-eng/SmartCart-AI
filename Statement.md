SmartCart AI – Shopping Cart with Recommendation System

## Problem Description
In everyday life, people often face difficulties in managing their shopping items and calculating the total bill manually. This process can be inefficient and may lead to errors.

There is a need for a simple system that can:
1. Manage shopping items easily
2. Automatically calculate total cost
3. Provide helpful suggestions to users

## Objective
The objective of this project is to develop a shopping cart system using C++ that:
1. Allows users to add, remove, and update items
2. Calculates total bill with GST
3. Provides basic intelligent recommendations

## Proposed Solution
To solve this problem, a menu-driven program is created that:
1. Displays available items
2. Allows users to manage their cart
3. Generates a final bill
4. Suggests related items using simple logic

## AI Relevance
The project includes a basic rule-based recommendation system, which is a simple form of Artificial Intelligence. It improves user experience by suggesting related items.

## Conclusion
This project provides a simple and effective solution for shopping management while introducing basic AI concepts.
